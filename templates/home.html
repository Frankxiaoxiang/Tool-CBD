Tool cost Database

This tool is designed to collect, store, compare, and analyze tool cost breakdown data.

> The input data can be saved to a database or exported as a CSV file.

> You can load and compare cost data from different CSV files to clearly identify differences between suppliers.

> Currently, only plastic injection and tooling cost databases are available; other sections will be available later.
